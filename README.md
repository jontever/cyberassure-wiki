# CyberAssure — Security Architecture Wiki

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/jontever/cyberassure-wiki)

An open-source security architecture reference wiki covering enterprise network security, zero trust, threat modelling, identity, data protection, and more.

🌐 **Live site:** [cyberassure.uk](https://cyberassure.uk)

## Topics covered

- Secure system design, DARIOM lifecycle & post-quantum cryptography
- Zero trust principles & architecture
- Threat modelling with MITRE ATT&CK, ransomware & frontier AI threats
- Physical, Layer 2 & Layer 3 security
- Network segmentation (macro, micro, identity-based)
- Hardening hybrid infrastructure
- NGFW, NSM, NIDS/NIPS
- VPNs, ZTNA & SASE
- WAF, database firewalls & data-centric security
- Privileged access & identity federation
- IoT security & endpoint/device management
- Security logging, AI, analytics & SIEM
- Risk & governance: asset management, supply chain, NCSC CAF, Cyber Essentials, OPSEC, people-centred security, penetration testing
